Europa Universalis IV Savegame Stats

Performs a series of statistics on an uploaded savegame and then redirects to a permanent web page with these statistics. This service is ran off a low-end server in New York, and a savegame uploaded during off hours with a fast internet connection will take approximately 2-3 seconds

Steps:

  • Locate your local savegame
  • Compress your savegame to a zipped or gzipped file
  • Upload the compressed file
  • Wait

All the technology behind this website is free, open sourced, and is hosted on Github. If you have any improvements or concerns, please raise an issue on Github or post to the forums.

Europa Universalis IV Savegame Stats

Performs a series of statistics on an uploaded savegame and then redirects to a permanent web page with these statistics. A modern browser (IE 10+, Chrome, Firefox, etc) is required. This service is ran for free and without ads because this only costs me $5 a month and I can swallow that; however, the hardware can't take a large load, so please be aware of that.

Steps:

  • Locate your local savegame
  • Compress your savegame to a zipped or gzipped file
  • Upload the compressed file
  • Wait hopefully 2-3 seconds

All the technology behind this website is free, open sourced, and is hosted on Github. If you have any improvements or concerns, please raise an issue on Github or post to the forums.
